# Tweet-Stat
A little console program which prints stats about any twitter user.

## How to use
1. Create a new app on <https://apps.twitter.com/>
2. Click on __Test OAuth__ and copy the keys on the file __keys.py__
3. Run `python main.py`
4. Enjoy!
